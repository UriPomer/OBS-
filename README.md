# OBS+哔哩哔哩直播姬一键推流

## 制作原因
### 每次想在b站开直播给朋友看的时候都需要进行以下操作：
- 打开obs和直播姬
- 点击直播姬上的“添加素材”按钮
- 点击“第三方推流”按钮
- 点击obs上的“开始直播”按钮

    虽然只有短短四步，但是本人是个懒人，且想试着上手一下qt，于是做出了这个程序


## 使用语言
- 完全使用ahk和qt c++


## 使用方法
- 在setting.json中设置你的直播姬和obs的路径和程序名
- 设置startDelayTime参数：此参数单位为s，不接受小数点，根据你电脑完全打开直播姬的时间设定
- 设置obsDelayTime参数，此参数按理来说应该比前一项大，作用是延迟obs的启动，可自行调试
